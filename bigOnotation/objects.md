# Objects
## Unordered, key value pairs

```js
let tamu = {
    name: "Texas A&M University",
    location: "College Station, TX",
    mascot: "Aggies"
}
```

## When to use objects:
- When you don't need order
- When you need fast access or insertion/removal

## Big O of Objects:
- Insertion: O(1)
- Removal: O(1)
- Searching: O(n)
- Access: O(1)
- Object.keys: O(n)
- Object.values: O(n)
- Object.entries: O(n)
- hasOwnProperty: O(1)
